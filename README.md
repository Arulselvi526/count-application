## Counter Application – React Project

A modern Counter Application built with React.js that showcases the power of the useState Hook for managing dynamic UI updates. The app allows users to increment, decrement, and reset the counter instantly with a smooth and responsive interface.

### ✨ Key Highlights

⚛️ Built using React Functional Components

🔢 Real-time counter updates with useState

➕ Increment functionality

➖ Decrement functionality with minimum limit handling

🔄 Reset button to restore the initial state

🌙 Light/Dark mode toggle for better user experience

📱 Responsive and clean UI design

### 🛠️ Tech Stack
React.js
JavaScript
HTML5
CSS3
